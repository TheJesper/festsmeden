# Festsmeden

**The Party Smith** - A party planner application for coordinating events, voting on activities, and managing celebrations.

## Features

- Event voting system
- Activity coordination
- Party planning tools

## Quick Start

```bash
npm install
npm run dev
```

Dev server runs on port **9558**.

## MVP

The MVP voting component is in `mvp/svensexa-vote.jsx`.
